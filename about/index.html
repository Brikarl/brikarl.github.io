关于本博客

北京某高校材料硕士在读，自大二起在个人公众号上发表影评，在B站发布视频。现自学计算机相关知识准备转行。学习之路也是相当曲折，研一的时间基本花在自身专业课程学习上，基本只是看了 Python 相关知识以及研一下学期的「算法设计与分析」选修课上，最后课程也取得了还算不错的成绩，算是为自己转行增加了些信心。

正式学习还算从研二开始算起，研二上学期涉猎了 Python 的相关应用，比如 Flask Web 搭建、爬虫以及数据分析的知识，大多数也是浅尝辄止，了解些计算机知识真正落地的内容。后来在友人的建议下学习基础知识：研一上完成了组成原理和操作系统的学习。疫情延期开学期间完成了对前面知识的复习与巩固，3 月份开始准备实习以及其他知识的学习。

本博客部署在 GitHub 和 Coding 平台，目前的用途是在记录自己计算机学习的内容，其中大部分内容通过转载获得，仅作为记录，不作商业用途。

未来也会有书籍与电影的相关内容，更多的书影记录会发布在豆瓣上。也欢迎大家与我交流，我的联系方式可以在侧边栏中找到。
